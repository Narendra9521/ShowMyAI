Our commitment to security and privacy

Reporting Security Issues

We take the security of ShowMyAI seriously. If you believe you've found a security vulnerability in our website or services, we encourage you to notify us. We welcome working with security researchers to improve the security of our platform.

How to Report a Vulnerability

Please email security concerns to showmyai999@gmail.com with the following information:

  • Description of the vulnerability
  • Steps to reproduce the issue
  • Potential impact of the vulnerability
  • Any suggestions for mitigating the issue

Our Commitment

When you report a vulnerability to us, we commit to:

  • Acknowledge receipt of your report within 48 hours
  • Provide an estimated timeframe for addressing the vulnerability
  • Notify you when the vulnerability is fixed
  • Give proper credit to you for the discovery (unless you prefer to remain anonymous)

Scope

This security policy applies to all services and websites under the showmyai.com domain.
